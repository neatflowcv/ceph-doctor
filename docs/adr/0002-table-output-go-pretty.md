# ADR 0002: 테이블 출력 라이브러리 - go-pretty

날짜: 2026-02-23
상태: 채택

## 배경

CLI 출력에서 목록 정보를 표 형태로 제공해야 한다. 요구사항:

- 긴 문자열을 포함한 컬럼 출력의 가독성을 유지할 것.
- 컬럼 너비, 랩, 정렬 등 출력 표현을 세밀하게 제어할 수 있을 것.
- 향후 출력 포맷 확장이 가능할 것.

## 결정

테이블 출력 라이브러리로 go-pretty를 사용한다.

## 근거

- 컬럼 너비와 랩/정렬 옵션을 제공해 긴 문자열의 표현을
  깔끔하게 제어할 수 있다.
- 테이블 출력 외에도 다양한 포맷으로 확장할 수 있어
  CLI 출력 확장에 유리하다.

## 대안

- tablewriter: 널리 쓰이고 기능이 풍부하나 출력 표현의
  미세 조정 측면에서 go-pretty 대비 유연성이 떨어진다.
- rodaine/table: 단순하고 가볍지만 출력 포맷 확장성과
  세밀한 제어가 부족하다.

## 결과

- 테이블 출력이 필요한 커맨드는 go-pretty를 사용한다.
- 출력 스타일과 랩 전략은 실제 구현 단계에서 커맨드별로
  결정한다.
